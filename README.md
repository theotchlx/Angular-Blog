# Angular-Blog
A simple blog in Angular, with a pocketbase backend.  
Monorepo  

Angular config:  
Style chosen : CSS  
SSR/SSG: No  

### Features

- Backend (bonus)
- Authentication
  - Registering a user (bonus)
  - Logging in
  - Logging out
- Blogs
  - Blogs page
  - Creating blogs



### Launching the backend

Execute the following command:
```
you@pc:~/Angular-Blog$
> pocketbase serve
```
